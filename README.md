# Flight Delay & Cancellation Analysis

## Problem Statement
Analyze factors affecting flight delays and cancellations to improve airline operations.

## Dataset
1M+ flight records including taxi times, delays, and cancellations.

## Tools Used
- Python (Pandas, NumPy)
- Plotly
- Matplotlib

## Key Insights
- Internal delays > External delays -Late Aircraft delays are 4x more than weather delays  
- High congestion airports show higher cancellations, suggesting that ground congestion is a key operational bottleneck 
- Seasonal patterns observed, Peak Demand creates operational stress and higher disruption risk  

## Recommendations
- Optimize aircraft scheduling  
- Improve airport ground operations  
- Focus on peak periods
